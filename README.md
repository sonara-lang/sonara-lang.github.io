# sonara-lang.github.io

Landing page for [Sonara](https://github.com/sonara-lang/sonara-lang) — a text-based music composition language that compiles `.son` files to MP3/WAV.

## Stack

- React 18
- Vite 5
- Tailwind CSS 3

## Dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `../docs/` (GitHub Pages root).

## Deploy

Push to `main`. GitHub Pages serves from `docs/`.
